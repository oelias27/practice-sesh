# Practice Sesh

```WIP```

Practice-Sesh is an open-source project to help musicians structore and keep track of their practice sessions and practice patterns. It's built with React.js, Node.js, Express.js, and MongoDB.

The backend for Practice Sesh can be found here: <https://github.com/oelias27/practice-sesh-backend>
